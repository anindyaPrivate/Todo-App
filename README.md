# 📝 **Python To-Do App**

This is a simple To-Do application built with Python 3.12. The application allows users to add, view, and manage their to-do items.


**Features**

➕ Add new to-do items

👀 View existing to-do items

✅ Mark items as complete

**Requirements**

🐍 Python 3.12 or higher
